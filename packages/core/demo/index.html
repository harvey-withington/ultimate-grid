⚡ Ultimate Data Grid — Quick Reference

Row Selection

ActionResult
Click a rowSelect that row (deselects all others). Click again to deselect.
Ctrl + ClickToggle that row on/off without affecting other selected rows.
Click + dragDrag-select a contiguous range of rows.
Ctrl + drag on unselected rowAdd dragged range to existing selection.
Ctrl + drag on selected rowRemove dragged range from existing selection.
Two-finger touch dragDrag-select on mobile / trackpad.

Sorting

ActionResult
Click column headerSort by that column ascending (▲). Click again for descending (▼). Third click removes sort.
Ctrl + Click headerAdd as a secondary sort (multi-sort). The primary sort column is unchanged. Repeat to cycle asc → desc → off for that column independently.
Clear Sort buttonRemove all active sorts.

The sort priority order is shown in the stats bar, e.g. department asc, salary desc.

Filter Expressions

Type directly into any filter input below the column header. Expressions are parsed automatically — no need to select an operator from a dropdown.

ExpressionTypeMeaning
aliceTextContains "alice" (case-insensitive)
=EngineeringTextExactly equals "Engineering"
!=SalesTextDoes not equal "Sales"
^ATextStarts with "A"
on$TextEnds with "on"
>80000NumberGreater than 80 000
>=50000NumberGreater than or equal to 50 000
<60000NumberLess than 60 000
50000..80000NumberBetween 50 000 and 80 000 (inclusive)
=75 or 75NumberExactly 75
2020..2023DateJoined between 2020-01-01 and 2023-12-31
2021-06..2022-03DateJoined between June 2021 and March 2022
>2022-01-01DateJoined after Jan 1 2022
=2019-03-04DateJoined on exactly that date

A red border on a filter input means the expression could not be parsed.

Multiple column filters are combined with AND. Use the Clear Filters button to reset all at once.
