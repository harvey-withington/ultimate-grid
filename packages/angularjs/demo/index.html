angular.module('myApp', ['ultimateGrid'])

<!-- 3. Drop the directive in your template -->
<ultimate-grid
  column-defs="vm.columnDefs"
  row-data="vm.rowData"
  options="vm.gridOptions"
  selection-mode="multi"
  on-grid-ready="vm.onGridReady(api)"
  api="vm.gridApi">
</ultimate-grid>

<!-- 4. Listen to grid events -->
$rootScope.$on('ugrid:selectionChanged', function(_, e) {
  vm.selectedCount = e.selectedRowIds.length;
});

⚡ Ultimate Data Grid — Quick Reference

Row Selection

ActionResult
Click a rowSelect that row (deselects all others). Click again to deselect.
Ctrl + ClickToggle that row on/off without affecting other selected rows.
Click + dragDrag-select a contiguous range of rows.
Ctrl + drag on unselected rowAdd dragged range to existing selection.
Ctrl + drag on selected rowRemove dragged range from existing selection.
Two-finger touch dragDrag-select on mobile / trackpad.

Sorting

ActionResult
Click column headerSort ascending (▲). Click again for descending (▼). Third click removes sort.
Ctrl + Click headerAdd as a secondary sort (multi-sort).

Filter Expressions

Type directly into any filter input below the column header.

ExpressionTypeMeaning
aliceTextContains "alice" (case-insensitive)
=EngineeringTextExactly equals "Engineering"
!=SalesTextDoes not equal "Sales"
^ATextStarts with "A"
on$TextEnds with "on"
>80000NumberGreater than 80 000
50000..80000NumberBetween 50 000 and 80 000
2020..2023DateJoined between 2020 and 2023
>2022-01-01DateJoined after Jan 1 2022

Multiple column filters are combined with AND. Active filters appear in the stats bar.
